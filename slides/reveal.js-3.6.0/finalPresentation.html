Cascalog: Big Data


Kushal Cuttari, Avi Banerjee, Danil Kolesnikov, Jason Thai

Cascalog Overview:

  • Querying libary for Clojure or Java
  • Used for processing "Big Data" using Hadoop

Hadoop?

  • Java open source data analytics engine
  • Used for processing "Big Data" using Hadoop
  • Most use cases are data pipelines - functional in nature

MapReduce Paradigm

  • Allows parallelized distributed processing on large data sets
  • Converts data into sets of tuples
  • Tuples are further reduced and processed

Our Project

  • Use Cascalog to query airline data from 2012-2017 to find percentage of delayed flights
  • Dataset includes over 130 million points
  • Use Hadoop to create taps into our CSV
  • Create Cascalog predicates that query and aggregate the data

Were We Successful?

  • Yes! Were able to gather insights about our data as well as model it over time

So...Cascalog?

  • Incredibly powerful with simple syntax and excellent predicate structure
  • Little online documentation and few examples makes first time development rely on lot of trial and error
  • Terminal error statements are almost impossible to understand
  • Hadoop is hard to configure and took many hours to get working
